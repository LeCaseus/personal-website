why from scratch?

Most personal sites get bootstrapped from a template or a framework — which is totally fine, but I wanted to actually understand what I was building. Writing the CSS by hand meant learning why things break. Writing the server from scratch meant understanding what a route actually is.

This site is less about the end result and more about the process of building it. It'll keep changing as I learn new things, and that's kind of the point.
