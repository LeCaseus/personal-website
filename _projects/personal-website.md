description: "This site started as a microblogging/blogging platform I made for myself to share my thoughts instead of scouring through different platforms to find the right audience. The original version however, had a bunch of backend like Node.js, Express, and a PostgreSQL database, that was just part of my vibe-coding shenanigans, and I never really understood what I did. So rather than carry forward with code I couldn't explain, I stripped it all back to zero and rebuilt using only HTML, CSS, and vanilla JavaScript. It's hosted on GitHub Pages with no build step, no bundler, and no dependencies beyond Google Fonts and Font Awesome. Each section is on one plain HTML file. The philosophy is deliberate: complexity gets added back only once I understand what it does and why it's needed. The site has now become a portfolio, a blog, and a personal learning log, all in one."
