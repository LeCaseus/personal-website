# chezter*vargas* — personal website

A personal portfolio and micro-blog website built from scratch using HTML, CSS, JavaScript, and Node.js.

---

## what's inside

- **Home** — hero section, portfolio highlights, latest blog posts, and a notes feed
- **Blog** — long reads on personal life, gaming, and career
- **Notes** — a masonry feed of short notes, letters, and thoughts
- **Portfolio** — selected projects, built things, and work
- **Resume** — skills, experience, and education
- **About** — who I am and what I'm up to
- **Contact** — ways to reach me

---

## built with

- HTML, CSS, Vanilla JavaScript
- Node.js + Express
- PostgreSQL via [Neon](https://neon.tech)
- Deployed on [Vercel](https://vercel.com)
- Fonts: [Fraunces](https://fonts.google.com/specimen/Fraunces) + [DM Sans](https://fonts.google.com/specimen/DM+Sans)
